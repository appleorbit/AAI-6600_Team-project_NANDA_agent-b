# Cost of Living Calculator — Agent B

**AAI 6600 | Northeastern University | NANDA Society of Agents**
**Developed by Wei Dong**

Agent B is a standalone AI agent that evaluates the cost of living for any given city and checks whether a job's salary is enough to live comfortably there. It is part of a larger multi-agent system built on the NANDA sandbox, where it works alongside Agent A (Job Scout) to provide financial context for every job listing.

## What It Does

Agent B takes a job title, city, and salary range as input, then returns:

- **Monthly salary range** — converts annual or hourly salary into monthly figures
- **Monthly living cost** — rent + food + commute + necessities for that city
- **Monthly surplus** — how much money is left after living costs
- **Financial status** — a rating based on the US 50/30/20 personal finance rule
- **AI comment** — a one-sentence insight generated by Google Gemini

### Financial Status Rating

Based on the ratio of monthly living cost to monthly salary:

| Status | Condition |
|--------|-----------|
| 🟢 Comfortable | Living cost is less than 40% of monthly salary |
| 🟡 Moderate | Living cost is 40–60% of monthly salary |
| 🔴 Tight | Living cost is more than 60% of monthly salary |

---

## How the Calculation Works

Using **Data Scientist in Boston, $80k - $100k** as an example:

**Step 1 — Salary Conversion**
```
$80,000 ÷ 12 = $6,667 / mo (min)
$100,000 ÷ 12 = $8,333 / mo (max)
Midpoint = $7,500 / mo
```

**Step 2 — Living Cost**
```
Rent:        $2,000 - $2,800 / mo
Food:        $600 / mo
Commute:     $90 / mo
Necessities: $200 / mo
─────────────────────────────────
Total:       $2,890 - $3,690 / mo
Midpoint:    $3,290 / mo
```

**Step 3 — Financial Status**
```
Ratio = $3,290 ÷ $7,500 = 44%
40%–60% range → 🟡 Moderate
```

**Step 4 — Monthly Surplus**
```
Min surplus = $6,667 - $3,690 = $2,977 / mo
Max surplus = $8,333 - $2,890 = $5,443 / mo
```

## API Endpoints

### POST `/api/v1/evaluate`
Main endpoint. Called by Agent A (Job Scout) to evaluate living costs for each job.

**Request:**
```json
{
  "job_title": "Data Scientist",
  "location": "Boston, MA",
  "estimated_salary": "$80k - $100k"
}
```

**Response (full):**
```json
{
  "status": "success",
  "city": "Boston",
  "monthly_salary_range": "$6,667 - $8,333 / mo",
  "monthly_cost_range": "$2,890 - $3,690 / mo",
  "monthly_surplus_range": "$2,977 - $5,443 / mo",
  "affordability": "🟡 Moderate",
  "ai_comment": "This Data Scientist role in Boston offers a moderate financial outlook.",
  "cost_breakdown": {
    "rent": "$2,000 - $2,800 / mo",
    "food": "$600 / mo",
    "commute": "$90 / mo",
    "necessities": "$200 / mo"
  },
  "for_agent_a": {
    "monthly_cost_range": "$2,890 - $3,690 / mo",
    "affordability": "🟡 Moderate"
  }
}
```

### GET `/api/v1/cities`
Returns the list of supported cities.

### GET `/health`
Health check endpoint.

## How to Run Locally

**1. Clone the repository**
```bash
git clone https://github.com/appleorbit/AAI-6600_Team-project_NANDA_agent-b.git
cd AAI-6600_Team-project_NANDA_agent-b
```

**2. Create a virtual environment**
```bash
python3 -m venv venv
source venv/bin/activate
```

**3. Install dependencies**
```bash
pip install fastapi uvicorn google-generativeai python-dotenv requests streamlit
```

**4. Create a `.env` file**
```
GOOGLE_API_KEY=your_google_api_key_here
```
Get a free API key at: https://aistudio.google.com

**5. Start the backend**
```bash
python main.py
```

**6. Start the frontend (in a new terminal)**
```bash
streamlit run frontend.py
```

**7. Open in browser**
- Frontend: http://localhost:8501
- API Docs: http://localhost:8083/docs

## How Agent A Calls Agent B

Agent B is designed to be called by Agent A (Job Scout) through Module D (LangGraph orchestrator). After Agent A finds job listings, Module D sends each job's details to Agent B's `/api/v1/evaluate` endpoint. Agent B returns two fields that are displayed on each job card in Agent A's frontend:

```json
{
  "monthly_cost_range": "$2,890 - $3,690 / mo",
  "affordability": "🟡 Moderate"
}
```

Agent B's public API endpoint for Agent A:
```
http://66.228.47.228:8083/api/v1/evaluate
```
