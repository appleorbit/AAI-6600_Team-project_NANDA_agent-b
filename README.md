# CostCompass — Cost of Living Calculator (Agent 3)

**AAI 6600 | Northeastern University | NANDA Society of Agents**  
**Developed by Wei Dong**

> **A note on naming:** During early development, this agent was designated **Agent B**, corresponding to the Agent A designation used for Job Scout. As the project progressed, the team adopted a unified numbering scheme, and Agent B was formally renamed **Agent 3**. Both names refer to the same agent.

CostCompass is a standalone AI agent that evaluates the cost of living in U.S. cities and determines whether a job's salary is sufficient for comfortable living there. It works alongside Job Scout (Agent 1) in the NANDA Society of Agents system, providing financial context for every job listing.

## What It Does

CostCompass takes a job title, city, and salary range as input, then returns:

- **Monthly salary range** — converts annual or hourly salary into monthly figures
- **Monthly living cost** — rent + food + commute + necessities for that city
- **Monthly surplus** — how much money is left after living costs
- **Financial status rating** — based on the U.S. 50/30/20 personal finance rule
- **AI comment** — a one-sentence insight generated by Google Gemini

### Financial Status Rating

| Status | Condition |
|--------|-----------|
| 🟢 Comfortable | Living cost is less than 40% of monthly salary |
| 🟡 Moderate | Living cost is 40–60% of monthly salary |
| 🔴 Tight | Living cost is more than 60% of monthly salary |

---

## How the Calculation Works

Using **Data Scientist in Boston, $80k – $100k** as an example:

**Step 1 — Salary Conversion**
```
$80,000 ÷ 12 = $6,667 / mo  (min)
$100,000 ÷ 12 = $8,333 / mo  (max)
Midpoint = $7,500 / mo
```

**Step 2 — Living Cost**
```
Rent:         $2,000 – $2,800 / mo
Food:         $600 / mo
Commute:      $90 / mo
Necessities:  $200 / mo
──────────────────────────────────
Total:        $2,890 – $3,690 / mo  |  Midpoint: $3,290 / mo
```

**Step 3 — Financial Status**
```
Ratio = $3,290 ÷ $7,500 = 44%
40%–60% range → 🟡 Moderate
```

**Step 4 — Monthly Surplus**
```
Min surplus = $6,667 – $3,690 = $2,977 / mo
Max surplus = $8,333 – $2,890 = $5,443 / mo
```

## API Endpoints

### POST `/api/v1/evaluate`
Main endpoint. Called by Job Scout (Agent 1) to evaluate living costs for each job.

**Request:**
```json
{
  "job_title": "Data Scientist",
  "location": "Boston, MA",
  "estimated_salary": "$80k - $100k"
}
```

**Response (full):**
```json
{
  "status": "success",
  "city": "Boston",
  "monthly_salary_range": "$6,667 - $8,333 / mo",
  "monthly_cost_range": "$2,890 - $3,690 / mo",
  "monthly_surplus_range": "$2,977 - $5,443 / mo",
  "affordability": "🟡 Moderate",
  "ai_comment": "This Data Scientist role in Boston offers a moderate financial outlook.",
  "cost_breakdown": {
    "rent": "$2,000 - $2,800 / mo",
    "food": "$600 / mo",
    "commute": "$90 / mo",
    "necessities": "$200 / mo"
  },
  "for_agent_a": {
    "monthly_cost_range": "$2,890 - $3,690 / mo",
    "affordability": "🟡 Moderate"
  }
}
```

### GET `/api/v1/cities`
Returns the list of supported cities.

### GET `/health`
Health check endpoint.

## How to Run Locally

**1. Clone the repository**
```bash
git clone https://github.com/appleorbit/AAI-6600_Team-project_NANDA_agent-b.git
cd AAI-6600_Team-project_NANDA_agent-b
```

**2. Create a virtual environment**
```bash
python3 -m venv venv
source venv/bin/activate
```

**3. Install dependencies**
```bash
pip install fastapi uvicorn google-generativeai python-dotenv requests streamlit
```

**4. Create a `.env` file**
```
GOOGLE_API_KEY=your_google_api_key_here
```
Get a free API key at: https://aistudio.google.com

**5. Start the backend**
```bash
python main.py
```
Backend runs at http://localhost:8083 — API docs at http://localhost:8083/docs

**6. Start the frontend (new terminal)**
```bash
streamlit run frontend.py
```
Frontend runs at http://localhost:8501

## How Job Scout Calls CostCompass

CostCompass is called by Job Scout (Agent 1) through the LangGraph Orchestrator (Module D). After Job Scout finds job listings, Module D sends each job's details to CostCompass's `/api/v1/evaluate` endpoint. CostCompass returns a `for_agent_a` payload with two fields displayed on each job card:

```json
{
  "monthly_cost_range": "$2,890 - $3,690 / mo",
  "affordability": "🟡 Moderate"
}
```

Public endpoint for Job Scout:
```
http://66.228.47.228:8083/api/v1/evaluate
```
